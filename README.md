# Prodigy_DS_001
Objective : Create a bar chart or histogram to visualize the distribution of a categorical or continuous variable.

Description : In this project, I performed basic data visualization using Python. I created a sample dataset containing the Age and Gender of individuals using the Pandas library. The dataset was converted into a DataFrame for analysis.

I visualized the data by creating : A Histogram to show the distribution of ages.
A Bar Chart (Count Plot) to display the distribution of genders.
The charts were customized with appropriate titles, labels, and colors to improve readability and presentation.

Technologies Used : 
Python
Pandas
Matplotlib
Seaborn

Outcome : This project demonstrates the fundamentals of Exploratory Data Analysis (EDA) by visualizing numerical and categorical data. It helps in understanding how data is distributed and how visualization techniques can be used to gain meaningful insights.
